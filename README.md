# running-schedule
러너들을 위한 마라톤 대회 일정 서비스

## 지원 플랫폼
- iOS, AOS, Web

## 컨벤션
### git commit convention
- feat: 새로운 기능 추가 비즈니스 로직 변경
- fix: 버그 수정
- docs: 문서 수정
- style: 코드 포맷팅, 세미콜론 누락 등의 스타일 수정
- refactor: 코드 리팩토링
- test: 테스트 코드, 리팩토링 테스트 코드 추가
- chore: 빌드 업무 수정, 패키지 매니저 수정
- build: 프로젝트의 빌드 시스템 수정
- ci: CI관련 설정 수정
- perf: 성능 개선
- revert: 작업 되돌리기
- temp: 임시 작업
- etc: 기타 변경사항
